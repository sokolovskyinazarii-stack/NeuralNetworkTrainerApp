# Neural Network Trainer C#

Лабораторна робота №3: **Робота з нейронними мережами. Навчання мережі**.

Проєкт реалізовано мовою C# як рішення з двох частин:

- `NeuralNetworkLab.Core` — бібліотечний модуль для створення, навчання та тестування нейронної мережі;
- `NeuralNetworkTrainerApp` — WinUI 3 / XAML-додаток для запуску експериментів і перегляду результатів.

## Функціональність

- генерація навчальної та тестової вибірки цифр 28x28;
- нормалізація пікселів у діапазоні 0..1;
- побудова мережі `784 -> Dense/ReLU -> Softmax`;
- зміна кількості нейронів у прихованому шарі;
- зміна кількості епох;
- додавання другого прихованого шару `Dense(64, ReLU)`;
- оцінювання точності на тестовій вибірці;
- виведення перших прогнозів;
- пошук першого прикладу, де мережа помилилася.

## Запуск

1. Відкрити `NeuralNetworkTrainerCSharp.sln` у Visual Studio.
2. Обрати проєкт запуску `NeuralNetworkTrainerApp`.
3. Запустити через `Ctrl + F5`.

## Збірка через командний рядок

```bash
dotnet build
```

## Примітка

Набір даних формується програмно як MNIST-подібна вибірка зображень цифр 28x28, щоб програма могла запускатися без великих зовнішніх файлів.
